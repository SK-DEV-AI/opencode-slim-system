Execute a command in a persistent shell.
Default CWD is the session worktree; use workdir, not cd chains.
Output truncates after ~2000 lines / 50KB — use Read or Grep on written-to temp file for full output.
Prefer Glob, Grep, Read, Edit, Write over cat, find, grep, sed in bash.
Never alter commit history (amend, force-push, hard reset). Never commit secrets.
Use the first user-provided SSH key for git push.
